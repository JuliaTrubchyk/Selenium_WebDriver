Selenium WebDriver Java – Practice Automation

A small project to practice UI test automation using Selenium WebDriver, JUnit 5, and Maven.

Tech Stack:

Java 21.0.6

Selenium 4.x (e.g., 4.38.0)

JUnit Jupiter (JUnit 5)

Apache Maven 3.9.11

IntelliJ IDEA

Notes:

This repo is for learning purposes. PRs with suggestions are welcome.
